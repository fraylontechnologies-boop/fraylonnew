## Image Specifications:

- **Format**: JPG, PNG, or WebP
- **Dimensions**: 400x200px (2:1 aspect ratio)
- **File Size**: Under 500KB for optimal loading
- **Style**: Professional, modern, relevant to each service category

## Recommended Image Types:

1. **No Code Development**: Screenshots of drag-and-drop interfaces, website builders
2. **Custom Development**: Code editors, development tools, programming languages
3. **Design Services**: Design tools, UI/UX mockups, creative workspace
4. **Marketing Services**: Analytics dashboards, social media graphics, marketing tools
5. **IT Solutions**: Server rooms, cloud infrastructure, technology consulting
6. **AI & Data Science**: AI/ML visualizations, data charts, neural networks
7. **Mobile App Development**: Mobile devices, app interfaces, development tools
8. **Zero to One**: Startup concepts, MVP development, rapid prototyping

## How to Add Images:

1. Create the directory structure above
2. Add your images with the exact filenames listed
3. The images will automatically appear in the service categories
4. If you want to use different filenames, update the `image` property in `src/pages/Services.tsx`

## Alternative: Use Online Images

You can also use online images by updating the image paths in the code:

```javascript
image: "https://your-domain.com/images/services/no-code-development.jpg"
```
